## Static Site Generators

Static site generators are tools that generate static websites from source files. They are popular among developers because they are simple to use, fast, and secure. Static sites are also easy to host and scale, and they are more secure than dynamic sites.

- [Jekyll](https://jekyllrb.com/)
- [Hugo](https://gohugo.io/)
- [Gatsby](https://www.gatsbyjs.com/)
- [Next.js](https://nextjs.org/)
- [VuePress](https://vuepress.vuejs.org/)
- [Nuxt.js](https://nuxtjs.org/)
- [Docusaurus](https://docusaurus.io/)
- [Gridsome](https://gridsome.org/)
- [11ty](https://www.11ty.dev/)
- [Pelican](https://blog.getpelican.com/)
- [Hexo](https://hexo.io/)

- [MkDocs](https://www.mkdocs.org/)
